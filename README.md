# GambCalc

A card game money calculator. Track wins, losses, and chip counts across multiple games with friends.

## Games

### Cat Te
Vietnamese card game with streak-based scoring. Win streaks multiply the pot — the longer you win, the more everyone pays. Features include normal wins, instant wins (x2), and visual effects that escalate with streak length.

### Poker
Chip-based poker with 3 betting rounds. Each player starts with 50 chips. Place bets, raise (max 5), call, or fold across 3 rounds before showdown. Buy more chips anytime if you run out.

## Features

- Undo / Redo with full history
- Per-player detailed breakdown (tap any player to see round-by-round results)
- Match history with soft delete and trash bin
- Optional cloud sync with user accounts
- Works offline without an account
- Remember me on login
- Mobile-first design with dark theme
